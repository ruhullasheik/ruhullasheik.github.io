career arc

The story so far

ACT I 2014 – 2020

🔗 The Integration Engineer

Six years inside SAP HANA, Oracle EBS, and JDE — owned complete integration projects for enterprise customers (SEI/7-Eleven, ATK, VISTA) solo, including direct customer communication. Built the pipelines that extracted, cleansed, and modelled supply chain data so enterprise planning systems could actually think.

Learned that owning a problem end-to-end — from customer call to production — is the fastest way to understand it deeply.

ACT II 2020 – 2024

☁️ The Platform Builder

Moved from integration templates to building the platform itself. Core member of the R&D team that re-platformed supply chain tooling into multi-tenant cloud services on Azure — Apache Airflow with OAuth2 SSO, Google Zanzibar authorization, IdentityServer4 identity gateway, Java/Quarkus middleware. 79% commit share on the Airflow service. Led the Platform Entitlements service as Tech Lead from day one.

Learned that platform engineering is about making other teams move faster — the best infra is invisible.

ACT III 2024 – now

🤖 The AI Platform Engineer

Building the infrastructure layer that lets AI agents act on real enterprise data. Led and built a production MCP (Model Context Protocol) gateway that auto-discovers OpenAPI specs and exposes them as tools — 20K+ API calls/day across a multi-tenant Azure platform. Also shipped a Langflow + Azure OpenAI conversational agent for natural-language tool discovery. Using Claude Code and spec-driven development throughout.

Still learning. Moving fast. The gap between what AI can do and what enterprises are actually using — that's the interesting problem.
